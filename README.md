# 栗帆个人作品集网站

这是栗帆的个人作品集网站，展示新媒体运营、品牌运营、海外增长运营、项目统筹、活动策划与内容制作相关经历。

在线预览：

https://lifan54a.github.io/lifan-portfolio/

## 内容结构

- 个人经历：武汉米创科技有限公司、深圳市阳林合科技有限公司相关履历与核心成果。
- 精选项目：海外文旅品牌、网红中国行、多平台矩阵运营、大型活动策划与执行。
- 个人优势：品牌从 0 到 1、新媒体矩阵运营、短视频全流程、数据化增长、项目统筹落地、AI 应用与创新。
- 作品展示：围绕品牌内容、视觉物料、短视频和项目执行沉淀的作品内容。
- 联系方式：提供电话、邮箱、微信与二维码交互展示。

## 技术栈

- React
- Vite
- pnpm
- GSAP / React Bits 视觉动效
- GitHub Pages

## 本地运行

```bash
pnpm install
pnpm dev
```

默认本地预览地址：

```text
http://127.0.0.1:5173/
```

## 构建

```bash
pnpm build
```

构建产物会输出到 `dist/`。
